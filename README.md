<h1 align="center">How to Use 🤔</h1>

````
- Clone this repository:
$ https://github.com/KratosOfficial/Youtube-Redesign

- Enter in directory:
$ cd YoutubeRedesign

- For install dependencies:
$ flutter packages get

- Run the app: 
$ flutter run

````
